# アプリの説明

Djangoで作成したクイズアプリです。  

---

## 主な機能

- クイズ作成機能
- クイズ編集機能
- クイズ検索機能
- クイズ回答機能

---

## 使用技術

- Python
- Django
- PostgreSQL(予定)
- Docker

---

## ブランチ運用ルール

- main　(本番用)
- feature/*　（機能開発用）
- fix/* (既存機能修正)

---

## コメントルール

### git commitの場合

- 【作成】新規作成
- 【途中】作成途中
- 【完成】作成完成
- 【修正】一度完成した機能の修正
- 【バグ】バグの修正

### ソースコードの場合

- NOTE: 設計理由、今後の展開
- TODO: 対応する作業
- FIXME: 明確な不具合
- HACK: やむを得ない実装
- WARNING: 触ると危険
